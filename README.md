# CNN Development on Custom Dataset

## Student ID
23-50308-1

## Project Overview
This project implements a Convolutional Neural Network (CNN) for classifying sign language images using the Sign Language MNIST dataset.

Two models were built:
- CNN without Batch Normalization and Dropout
- CNN with Batch Normalization and Dropout

The final model was selected based on validation performance.

## Files
- CNN_CVPR_23_50308_1.ipynb → Full notebook
- best_sign_language_cnn.pth → Trained model weights

## Techniques
- PyTorch
- CNN
- Adam Optimizer
- Cosine Annealing LR
- CrossEntropyLoss
